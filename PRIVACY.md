# 2. Information We Collect

## 2.1 Discord Authentication Information

When you sign in using Discord OAuth, the Service receives information authorized by Discord, including:

-   Discord User ID
-   Username
-   Global display name
-   Avatar
-   OAuth session information

The Service never receives, stores or has access to your Discord password.

## 2.2 Generated Embed Data

When you generate an embed, the Service may temporarily store:

-   Message content
-   Embed titles
-   Descriptions
-   Fields
-   Buttons
-   Images
-   Colors
-   Authors
-   Footers
-   Generated code
-   Expiration timestamp

Embed Data is stored solely to provide temporary retrieval using generated codes.

Embed Data is processed automatically and is not routinely reviewed by the Team except where reasonably necessary to operate, secure or comply with legal obligations.

Generated codes are public identifiers intended solely to retrieve temporary Embed Data.

Generated codes are **not** passwords, authentication credentials or security tokens.

## 2.3 Technical Information

The Service may automatically collect technical information reasonably necessary to operate and protect the Service, including:

-   IP addresses or portions thereof where reasonably necessary
-   Browser information
-   User agent
-   Request timestamps
-   Error logs
-   Security logs

This information is used exclusively for operational, security and abuse-prevention purposes.
